# Habit OS

Habit OS is a modern, gamified habit tracker designed to help you build and maintain positive routines. With a clean, high-hierarchy dashboard, it turns habit tracking into an engaging experience.

## Features

- **Gamified Tracking**: Earn points and level up as you complete your daily habits.
- **AI Assistant**: A built-in AI assistant to provide insights and keep you motivated.
- **Local Data Management**: Your data stays yours. Easily export and import your habits, progress, and profile data via JSON.
- **Modern UI**: Built with a sleek, responsive design using React, TailwindCSS, and shadcn/ui.

## Getting Started

This project is a monorepo powered by Turborepo. 

### Prerequisites

Make sure you have [Bun](https://bun.sh/) installed.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/mekanhaji/habit-os.git
   cd habit-os
   ```

2. Install dependencies:
   ```bash
   bun install
   ```

3. Start the development server:
   ```bash
   bun run dev
   ```

## Tech Stack

- **Framework**: React / Vite
- **Styling**: Tailwind CSS & shadcn/ui
- **Tooling**: Turborepo & Bun
